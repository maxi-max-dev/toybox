# 🧸 玩具盒 · Toybox

46 个一眼就懂、点开就能玩的小玩具。物理、绘画、氛围、小游戏，一个接一个搓出来的。

每一个都是**独立的单文件 HTML**，不联网、不装任何依赖，用浏览器打开就能玩。

### 👉 在线玩：**[https://maxi-max-dev.github.io/toybox](https://maxi-max-dev.github.io/toybox/)**

### 本地玩

```bash
git clone https://github.com/maxi-max-dev/toybox.git
# 双击任意 t/*.html，或直接打开 index.html
```

> 全部单文件、零构建、零依赖。`index.html` 是总入口，`t/` 下是 46 个玩具本体。
